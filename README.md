# gql-chat-client-ember

Example client for a chat app using GraphQL subscriptions with Ember and ember-apollo-client.

Assessment: non-subscription GraphQL works fine with ember-apollo-client, but `subscribeToMore` is not well-documented.

Not deployed to production.

## Requirements

- Node
- Yarn
- Ember CLI

## Installation

```
$ yarn install
```

## Running

```
$ ember serve
```
